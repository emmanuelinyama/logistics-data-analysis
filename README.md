# Logistics Data Analysis Project

## Overview
This project demonstrates SQL-based data analysis using a relational database designed for a logistics and delivery management system.

The database includes entities such as:
- Customer
- Orders
- Shipment
- Package
- Invoice
- Payment
- Branch
- Employee
- Delivery_Segment
- Route
- Vehicle

## Objectives
- Analyse shipment volumes
- Track payment status
- Identify high-performing branches
- Generate operational insights using SQL queries

## Tools Used
- MySQL
- SQL
- phpMyAdmin
- Relational Database Design (3NF)

## Sample Business Questions Answered
- How many shipments are processed per branch?
- Which customers generate the highest revenue?
- What is the average shipment value?
- Which invoices remain unpaid?

---

This project demonstrates practical database querying and analytical thinking suitable for entry-level Data Analyst roles.
